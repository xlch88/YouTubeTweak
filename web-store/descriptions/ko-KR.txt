YouTube Tweak은 파워 유저를 위한 YouTube 향상 확장 프로그램입니다. 다양한 맞춤 기능으로 기본 경험의 한계를 벗어나 세부까지 정밀 제어하며, 더 깔끔하고 똑똑하고 부드러운 시청 환경을 제공합니다.

⭐완전 무료, 오픈 소스!
⭐새 기능 추가나 번역 기여 환영!
👉 https://github.com/xlch88/YoutubeTweak

주요 기능:

🎬 플레이어 향상
화질 고정: 매 재생 시 지정한 화질(예: 1080p/720p)로 자동 전환, 없으면 가장 가까운 낮은 화질로 다운.
추가 재생 속도 버튼: 1.25x, 1.5x, 2x 등 추가 속도 버튼 표시.
채널별 속도 기억: 채널마다 속도 선호도를 따로 저장.
특정 버튼 숨기기: 필요한 거의 모든 플레이어 버튼을 숨겨 더 깔끔한 UI 제공.
엔드스크린 추천 숨기기: 영상 끝에 나오는 흐릿한 추천(추천 영상이나 크리에이터 안내)을 숨김.
볼륨 부스트: 라우드니스 노멀라이즈를 꺼서 최대 볼륨에서 진짜 100% 구현.

💬 댓글
닉네임 표시: 댓글 작성자의 채널 닉네임을 표시해 단순 사용자 이름 대신 보여줌.
댓글 내용 강화: 긴 댓글을 자동 펼치고 댓글 텍스트를 자동 번역.

🧹 기타 기능
홈 레이아웃 제어: 홈 추천 영상의 행당 개수를 고정해 일정한 그리드를 유지.
광고 차단(실험적): 광고 영상을 자동 건너뛰고, 썸네일/오버레이 광고를 숨김; 크리에이터 상품 홍보(티셔츠, 머치 등)도 차단.

⚙️ 일반 설정
설정 가져오기/내보내기: 한 번의 클릭으로 설정을 내보내고, 가져오고, 초기화하여 손쉽게 이전.
다국어 지원: 인터페이스가 여러 언어를 지원하며 번역이 지속적으로 업데이트됨.

🔧 개발 현황
이 확장은 활발히 개발 중입니다. 피드백과 기능 요청을 환영합니다. 아이디어가 있거나 버그를 발견했거나 기여하고 싶다면 GitHub를 방문하세요.
