## 🥰 关于此扩展

此浏览器扩展专为 **YouTube** 用户设计，旨在优化使用体验、修复官方设计中的不便，并为高级用户提供更多自定义功能。
项目采用 **MIT 开源协议**，源码完全公开透明，任何人都可以自由查阅、修改或二次开发。

我们郑重承诺：

- **不进行任何形式的商业化**，无广告、无付费解锁、无增值服务；
- **不嵌入任何追踪代码**，不会收集、分析或上传用户数据；
- **不获取任何隐私信息**，所有设置、缓存与偏好均仅保存在本地浏览器中；
- **尊重用户体验与自由**，所有功能均可手动启用或关闭，完全由用户自行掌控。

我们的目标很简单——让 YouTube 回归纯粹的视频观看体验，而不是广告和算法的游乐场。

您可以在扩展商店查看此扩展的功能列表，或者直接安装体验！
