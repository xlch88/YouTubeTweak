# 更新日志

[中文](/docs-i18n/CHANGELOG_CN.md) | [English](/CHANGELOG.md)

所有的Release版本的更新记录在此文件中。

如果您有问题、BUG反馈或功能建议，请前往 [GitHub问题页面](https://github.com/xlch88/YouTubeTweak/issues) 提交。

## [1.0.9] - 2025-11-22

## Fixed

- **播放器**: 修复时间悬浮标签在直播模式下显示的问题。
- **评论翻译**: 相似语言（例如 en_US、en_UK）不会被重复翻译。

### Added

- **检查更新**: 点击弹出窗口中的版本号以手动检查更新。
- **评论翻译**: 添加评论翻译功能的目标语言选择选项。
- **评论翻译**: 添加手动触发翻译的按钮。
- **播放器功能按钮**: 添加了两个新功能按钮：镜像视频 和 旋转视频。

## [1.0.8] - 2025-11-14

### 更新

- 每行视频数量: 将`每行视频数量`选项的最大值从`10`提升至`15`。

### 添加

- 视频进度条: 添加始终在视频下方显示进度条
- 视频时间: 添加在左下角显示视频`当前时间/长度`

## [1.0.7] - 2025-11-12

### 添加

- 播放速度按钮: 添加了更多选项 `(2.25x, 2.5x)` 并重写了相关UI。
- 隐藏Shorts: 勾选在首页隐藏Shorts后，同时会隐藏`搜索结果中的Shorts视频`。
- 广告屏蔽: 勾选“屏蔽频道主商品推荐”后会隐藏`播放器`里的`“推荐商品”悬浮按钮`。
- 广告屏蔽: 勾选“屏蔽频道主商品推荐”后会隐藏视频简介下方 `演出门票信息`。

### 修复

- 显示评论昵称: 修复了在`暗色模式`下评论昵称颜色异常的问题。
- 隐藏Shorts: 修复了在播放页面的`推荐视频`里仍然有Shorts的问题。
- 弹出窗口: 修复了在某些浏览器下`弹出窗口`宽度异常的问题。
